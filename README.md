📚 Java ile Ders Kayıt Ekranı

Bu proje, Java kullanılarak geliştirilmiş bir ders kayıt ekranı uygulamasıdır.
Kullanıcıların ders bilgilerini girip kayıt oluşturabildiği temel bir masaüstü (GUI) uygulamasıdır.

🚀 Özellikler

🧑‍🎓 Öğrenci bilgisi giriş alanı
📖 Ders seçimi / ders bilgisi ekleme
📝 Form tabanlı veri girişi
✅ Kayıt oluşturma işlemi
🎨 Java GUI arayüzü (.form yapısı mevcut)

🛠️ Kullanılan Teknolojiler

Java

Java Swing (Form Designer)

Nesne Yönelimli Programlama (OOP)

Ant (build.xml varsa eklenebilir)
